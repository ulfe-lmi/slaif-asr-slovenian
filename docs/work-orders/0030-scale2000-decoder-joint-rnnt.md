# Work Order 0030: Scale-2000 Audio-Conditioned Decoder+Joint RNNT Diagnostic

Status: in progress

This work order authorizes one `DIAGNOSTIC_ONLY` synthetic-only experiment on
the accepted scale-2000 augmented audio bank.

## Scope

- Train exactly one arm: `scale2000_augmented_decoder_joint_rnnt`.
- Start from the untouched `nvidia/nemotron-3.5-asr-streaming-0.6b`
  checkpoint.
- Use audio-conditioned RNNT loss on the existing scale-2000 augmented
  schedule.
- Train only `model.decoder` and `model.joint`.
- Keep the acoustic encoder, preprocessor, prompt pathway, tokenizer, and all
  adapter/text-only paths frozen or absent.
- Evaluate only with the noncanonical directional batch-32 suite.

Training must use exactly one visible NVIDIA GeForce RTX 2080 Ti. The second
RTX 2080 Ti may be reported and may be used only for optional independent

scale-2000 joint-adapter metrics. `accepted_parent` remains `none` regardless
of metrics. A positive directional result requires a later canonical batch-1
work order before any acceptance discussion.
