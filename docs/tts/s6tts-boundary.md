# S6TTS Boundary

S6TTS is treated as an external lab-origin Slovenian TTS system for internal
synthetic diagnostic audio only.

The ASR repository may contain wrapper code, pinned configuration, aggregate
hashes, and privacy-safe admission evidence. It must not vendor S6TTS source,
runtime data, dictionaries, diphone resources, binaries, generated audio, local
manifests, or logs.

Generated S6TTS audio is synthetic TTS output. It is not real speech, not
Apache-2.0 repository content, not public-release-cleared audio, and not
training-eligible unless a later explicit work order authorizes its use.

Public distribution of S6TTS-generated audio, or a model trained on it, requires
a later provenance and release review.
