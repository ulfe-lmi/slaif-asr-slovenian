# Reviews and Audits

Store privacy-safe strategic review briefs, external audit summaries, and remediation matrices here.

Do not store:

- private model outputs;
- raw protected transcripts;
- credentials;
- model weights;
- unsupported claims of certification.

A review should distinguish:

- current-scope defect;
- missing test;
- documentation drift;
- future feature;
- rejected recommendation;
- release blocker.

Every accepted current-scope finding should point to a work order or pull request and closure evidence.
